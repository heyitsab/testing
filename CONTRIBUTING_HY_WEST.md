# 🦙 Ներդրում Ընելու Ուղեցոյց - Արեւմտահայերէն

## 🌟 Ընդհանուր Ակնարկ

Շնորհակալ ենք այս նախագիծին ներդրում ընելու ձեր հետաքրքրութեան համար։ Այս ուղեցոյցը պիտի օգնէ ձեզ հասկնալու, թէ ինչպէս կարող էք արդիւնաւէտ ներդրում մը ընել։

### ✨ Էմոջիներու Կարեւորութիւնը

**Շատ կարեւոր՝** Մենք խրախուսած ենք էմոջիներու օգտագործումը ամէն տեղ որ հնարաւոր է։ Էմոջիները կը դարձնեն մեր հաղորդակցութիւնը աւելի պարզ, ընկերական եւ հետաքրքրական։ 🎉

#### 🎯 Ինչու Էմոջիներ

- 😊 **Հաղորդակցութիւն**: Էմոջիները կը յստակացնեն ձեր մտադրութիւնները
- 🎨 **Տեսողականութիւն**: Կը դարձնեն փաստաթուղթերը աւելի գրաւիչ
- 🔍 **Արագ Գտնելու Համար**: Կը հեշտացնեն կարեւոր տեղեկութիւններու գտնելը
- 💡 **Ստեղծագործական Արտայայտութիւն**: Կը թոյլ տան աւելի լաւ արտայայտել ձեր գաղափարները

## 🚀 Սկսելու Համար

### 📋 Նախապայմաններ

Մինչ ներդրում ընելը, համոզուեցէք որ ունիք հետեւեալները՝

- ✅ Ruby տեղադրուած է ձեր համակարգին վրայ
- ✅ Git-ի հիմնական իմացութիւն
- ✅ GitHub հաշիւ մը
- ✅ Էմոջիներ օգտագործելու էնտուզիազմ! 🎊

### 🔧 Նախագիծը Կարգաւորել

```bash
# Նախագիծը պատճէնել (clone)
git clone https://github.com/heyitsab/testing.git

# Թղթապանակին մէջ մտնել
cd testing

# Նոր ճիւղ մը ստեղծել (օգտագործէք էմոջի մը ձեր ճիւղին անունին մէջ!)
git checkout -b feature/✨-your-feature-name
```

## 📝 Ծրագրային Ապահովման Ոճ

### 💎 Ruby Ծրագրաւորման Յանձնարարութիւններ

Երբ կը գրէք կոդ, խնդրեմ հետեւեցէք այս ուղեցոյցներուն՝

- 🎯 **Պարզ եւ Կարդալի**: Գրեցէք պարզ կոդ որ հեշտ է հասկնալ
- 📖 **Մեկնաբանութիւններ**: Աւելցուցէք էմոջիներ ձեր կարեւոր մեկնաբանութիւններուն մէջ
- 🧪 **Փորձարկումներ**: Միշտ աւելցուցէք փորձարկումներ ձեր նոր յատկութիւններուն համար

```ruby
# ✅ Լաւ օրինակ - էմոջիներով մեկնաբանութիւն
def feed(food)
  # 🍽️ Ալպական կ'ուտէ տրուած կերը
  "Alpaca #{@name} is eating #{food}."
end

# 🎨 Լաւագոյն օրինակ - էմոջիներով եւ պարզ կոդ
def make_sound
  # 🔊 Ալպական իր յատուկ ձայնը կ'արձակէ
  "Alpaca #{@name} says 'Mmmm!'"
end
```

## 🔀 Pull Request-ներու Ուղեցոյց

### 📤 Pull Request Ներկայացնելիս

Երբ պատրաստ էք ձեր աշխատանքը կիսելու համար՝

1. 🔍 **Վերստուգեցէք Ձեր Կոդը**: Համոզուեցէք որ ամէն ինչ կը գործէ
2. ✅ **Փորձարկումներ Անցուցէք**: Վազեցուցէք բոլոր փորձարկումները
3. 📝 **Գրեցէք Նկարագրութիւն**: Օգտագործեցէք էմոջիներ ձեր PR անունին եւ նկարագրութեան մէջ

#### 🎯 PR Անուն Օրինակներ

```
✨ Աւելցնել նոր յատկութիւն ալպականերու համար
🐛 Ուղղել սխալ wool_length հատկութեան մէջ
📚 Թարմացնել փաստաթղթերը
🎨 Բարելաւել կոդին ձեւը
♻️ Վերափոխել Alpaca class-ը
```

#### 📋 PR Նկարագրութեան Կաղապար

```markdown
## 🎯 Նպատակ

[Նկարագրեցէք ի՞նչ կ'ընէ այս փոփոխութիւնը]

## 🔧 Փոփոխութիւններ

- 📌 [Փոփոխութիւն 1]
- 📌 [Փոփոխութիւն 2]
- 📌 [Փոփոխութիւն 3]

## ✅ Ստուգելու Ցուցակ

- [ ] 🧪 Փորձարկումները կ'անցնին
- [ ] 📖 Փաստաթղթերը թարմացուած են
- [ ] 😊 Էմոջիներ օգտագործուած են
- [ ] 🎨 Կոդը պարզ եւ կարդալի է

## 📸 Պատկերներ (եթէ կայ)

[Աւելցուցէք պատկերներ եթէ կան տեսողական փոփոխութիւններ]
```

## 🐛 Խնդիրներ Զեկուցելու Համար

### 📢 Նոր Issue Բանալիս

Երբ կը գտնէք սխալ մը կամ ունիք առաջարկ մը՝

1. 🔎 **Փնտռեցէք Առկայ Issues-ները**: Ստուգեցէք արդէն բանալուած չէ
2. 🎯 **Լինեցէք Յստակ**: Նկարագրեցէք խնդիրը մանրամասն
3. 😊 **Օգտագործեցէք Էմոջիներ**: Դարձուցէք ձեր issue-ն աւելի պարզ

#### 🏷️ Issue Անուններու Օրինակներ

```
🐛 Սխալ: Alpaca-ն չի կրնար ուտել երբ անունը դատարկ է
💡 Առաջարկ: Աւելցնել shear_wool() մեթոտ
📚 Փաստաթղթերը անհասկնալի են display_info-ի համար
🎨 Բարելաւել: Աւելի լաւ սխալի հաղորդագրութիւններ
❓ Հարց: Ինչպէ՞ս աշխատեցնել փորձարկումները
```

## 💬 Հաղորդակցութեան Ոճ

### 🤝 Խումբին Հետ Հաղորդակցիլ

Մենք կ'արժեւորենք յարգալից եւ ընկերական հաղորդակցութիւնը՝

- 😊 **Ընկերական Լինիլ**: Միշտ լինեցէք բարեացակամ եւ յարգալից
- 💭 **Կառուցողական Քննադատութիւն**: Տուէք օգտակար առաջարկներ
- 🎉 **Գնահատել Ուրիշներու Աշխատանքը**: Շնորհակալութիւն յայտնեցէք
- ✨ **Էմոջիներ Օգտագործել**: Դարձուցէք ձեր հաղորդագրութիւնները աւելի ջերմ

#### 💌 Հաղորդագրութիւններու Օրինակներ

```
👍 Շնորհաւոր! Այս փոփոխութիւնը շատ լաւն է!
💡 Հետաքրքիր գաղափար, բայց ինչպէ՞ս մասին եթէ նաեւ...
🤔 Կը մտածեմ որ կրնայինք բարելաւել...
🎯 Այս լուծումը կատարեալ է այս խնդիրին համար!
```
