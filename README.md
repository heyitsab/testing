# 🦙 Alpaca Class Project 🦙

Welcome to the **Alpaca Class Project**! 🎉 This is a simple Ruby project that models adorable alpacas with various attributes and behaviors. 🌟

## 📋 Features

✨ **Alpaca Attributes:**
- 📝 Name - Give your alpaca a unique name!
- 🎂 Age - Track how old your alpaca is
- 🎨 Color - Choose from a variety of colors
- 🧶 Wool Length - Measure that fluffy wool!

🎯 **Alpaca Behaviors:**
- 🍎 **Feed** - Give your alpaca some tasty food
- 🔊 **Make Sound** - Listen to your alpaca go "Mmmm!"
- ℹ️ **Display Info** - View all the details about your alpaca

## 🚀 Getting Started

### Prerequisites
- 💎 Ruby installed on your system

### Usage Example

```ruby
# Create a new alpaca 🦙
alpaca = Alpaca.new("Fluffy", 3, "white", 15)

# Feed your alpaca 🥕
puts alpaca.feed("carrots")
# => "Alpaca Fluffy is eating carrots."

# Hear your alpaca's sound 🎵
puts alpaca.make_sound
# => "Alpaca Fluffy says 'Mmmm!'"

# Check your alpaca's info 📊
puts alpaca.display_info
# => "Name: Fluffy, Age: 3, Color: white, Wool Length: 15 cm"
```
