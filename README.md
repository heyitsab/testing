# Ալպակա (Alpaca)

Սա պարզ Ruby նախագիծ է, որը իրականացնում է Ալպակա դաս:

## Նկարագրություն

Այս նախագիծը պարունակում է `Alpaca` դաս, որը ներկայացնում է ալպակա կենդանին: Դասը ունի հետևյալ հատկությունները և մեթոդները.

### Հատկություններ

- `name` - Ալպակայի անունը
- `age` - Ալպակայի տարիքը
- `color` - Ալպակայի գույնը
- `wool_length` - Ալպակայի բրդի երկարությունը (սմ-ով)

### Մեթոդներ

- `feed(food)` - Կերակրել ալպակային
- `make_sound` - Ալպական ձայն է հանում
- `display_info` - Ցուցադրել ալպակայի տեղեկատվությունը

## Տեղադրում

Համոզվեք, որ ունեք Ruby տեղադրված ձեր համակարգում:

```bash
ruby --version
```

## Օգտագործում

Ահա օրինակ, թե ինչպես օգտագործել `Alpaca` դասը.

```ruby
require_relative 'alpaca'

# Ստեղծել նոր ալպակա
my_alpaca = Alpaca.new("Պուշիկ", 3, "սպիտակ", 15)

# Ցուցադրել տեղեկատվությունը
puts my_alpaca.display_info
# Արդյունք: Name: Պուշիկ, Age: 3, Color: սպիտակ, Wool Length: 15 cm

# Կերակրել ալպակային
puts my_alpaca.feed("խոտ")
# Արդյունք: Alpaca Պուշիկ is eating խոտ.

# Ալպական ձայն է հանում
puts my_alpaca.make_sound
# Արդյունք: Alpaca Պուշիկ says 'Mmmm!'
```

## Թեստավորում

Նախագիծը ներառում է թեստային ֆայլ, որը ստուգում է կոդի որակը:

```bash
ruby alpaca_test.rb
```

Թեստը ստուգում է, որ կոդը չի պարունակում `puts` կամ `binding.pry` հայտարարություններ:

## Ֆայլեր

- `alpaca.rb` - Ալպակա դասի իրականացումը
- `alpaca_test.rb` - Թեստային ֆայլ
- `alpaca.jpg` - Ալպակայի նկար
