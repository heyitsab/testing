# Alpaca - Κλάση Ruby για Αλπάκα

## Περιγραφή

Αυτό το έργο είναι μια απλή κλάση Ruby που μοντελοποιεί ένα αλπάκα με διάφορα χαρακτηριστικά και συμπεριφορές. Το αλπάκα μπορεί να έχει όνομα, ηλικία, χρώμα και μήκος μαλλιού, και μπορεί να εκτελέσει διάφορες ενέργειες όπως το φαγητό και την έκδοση ήχων.

## Χαρακτηριστικά

Η κλάση `Alpaca` περιλαμβάνει τα ακόλουθα χαρακτηριστικά:

- **name** (όνομα): Το όνομα του αλπάκα
- **age** (ηλικία): Η ηλικία του αλπάκα σε έτη
- **color** (χρώμα): Το χρώμα του μαλλιού του αλπάκα
- **wool_length** (μήκος μαλλιού): Το μήκος του μαλλιού σε εκατοστά

## Μέθοδοι

Η κλάση παρέχει τις ακόλουθες μεθόδους:

### `initialize(name, age, color, wool_length)`
Δημιουργεί ένα νέο αντικείμενο Alpaca με τα καθορισμένα χαρακτηριστικά.

### `feed(food)`
Ταΐζει το αλπάκα με το καθορισμένο φαγητό και επιστρέφει ένα μήνυμα.

### `make_sound`
Κάνει το αλπάκα να εκδώσει τον χαρακτηριστικό του ήχο.

### `display_info`
Εμφανίζει όλες τις πληροφορίες για το αλπάκα σε μορφή κειμένου.

## Χρήση

Ακολουθεί ένα παράδειγμα χρήσης της κλάσης Alpaca:

```ruby
require_relative 'alpaca'

# Δημιουργία ενός νέου αλπάκα
my_alpaca = Alpaca.new("Φλάφι", 3, "καφέ", 15)

# Εμφάνιση πληροφοριών αλπάκα
puts my_alpaca.display_info
# Έξοδος: Name: Φλάφι, Age: 3, Color: καφέ, Wool Length: 15 cm

# Ταΐσμα του αλπάκα
puts my_alpaca.feed("χόρτο")
# Έξοδος: Alpaca Φλάφι is eating χόρτο.

# Ακούστε τον ήχο του αλπάκα
puts my_alpaca.make_sound
# Έξοδος: Alpaca Φλάφι says 'Mmmm!'

# Πρόσβαση και τροποποίηση χαρακτηριστικών
my_alpaca.wool_length = 20
puts "Νέο μήκος μαλλιού: #{my_alpaca.wool_length} cm"
```
