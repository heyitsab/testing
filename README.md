# 🦙 Alpaca Adventure! 🦙

Welcome to the most **adorable** Ruby project on the internet! 🎉✨

![Alpaca](alpaca.jpg)

## 🌟 What's This All About?

Ever wanted to manage your very own virtual alpaca farm? Well, now you can! 🎊 This delightful Ruby project lets you create, feed, and interact with fluffy alpacas! 🦙💕

## ✨ Features

🎨 **Colorful Alpacas** - Create alpacas in any color you can imagine!  
📏 **Wool Management** - Track your alpaca's wool length (because fashion matters!)  
🍎 **Feeding Time** - Feed your alpacas their favorite snacks!  
🔊 **Alpaca Sounds** - Listen to them say "Mmmm!" (it's adorable, trust us!)  
📋 **Info Display** - Get all the deets about your alpaca friends!  

## 🚀 Getting Started

### Prerequisites

Make sure you have Ruby installed! 💎

```bash
ruby --version
```

### 🎮 Quick Start

1. Clone this awesome repo! 📦
```bash
git clone https://github.com/heyitsab/testing.git
cd testing
```

2. Fire up your Ruby console! 🔥
```bash
irb -r ./alpaca.rb
```

3. Create your first alpaca! 🎨
```ruby
fluffy = Alpaca.new("Fluffy", 3, "white", 15)
```

## 💡 Usage Examples

### Creating an Alpaca 🦙

```ruby
# Create a fabulous alpaca!
sparkles = Alpaca.new("Sparkles", 2, "silver", 20)

# Display its info
puts sparkles.display_info
# => Name: Sparkles, Age: 2, Color: silver, Wool Length: 20 cm
```

### Feeding Your Alpaca 🍎🥕

```ruby
# Alpacas love treats!
puts sparkles.feed("carrots")
# => Alpaca Sparkles is eating carrots.

puts sparkles.feed("apples")
# => Alpaca Sparkles is eating apples.
```

### Hear the Sounds 🔊

```ruby
# Make your alpaca speak!
puts sparkles.make_sound
# => Alpaca Sparkles says 'Mmmm!'
```
