# アルパカプロジェクト

アルパカクラスを実装したシンプルなRubyプロジェクトです。

## 概要

このプロジェクトは、アルパカの基本的な特性と行動をシミュレートする`Alpaca`クラスを提供します。アルパカの名前、年齢、色、毛の長さなどの属性を管理し、餌やり、鳴き声、情報表示などの機能を持っています。

## 機能

- **アルパカの作成**: 名前、年齢、色、毛の長さを指定してアルパカを作成できます
- **餌やり**: アルパカに食べ物を与えることができます
- **鳴き声**: アルパカが鳴き声を出します
- **情報表示**: アルパカの詳細情報を表示します

## 使い方

### Alpacaクラスの基本的な使用例

```ruby
require './alpaca'

# アルパカを作成
my_alpaca = Alpaca.new("フワフワ", 3, "白", 15)

# アルパカに餌を与える
puts my_alpaca.feed("草")
# 出力: Alpaca フワフワ is eating 草.

# アルパカの鳴き声
puts my_alpaca.make_sound
# 出力: Alpaca フワフワ says 'Mmmm!'

# アルパカの情報を表示
puts my_alpaca.display_info
# 出力: Name: フワフワ, Age: 3, Color: 白, Wool Length: 15 cm
```

## テスト

プロジェクトには`alpaca_test.rb`というテストファイルが含まれています。このファイルは、コード内に`puts`文や`binding.pry`が含まれていないことを確認します。

テストを実行するには:

```bash
ruby alpaca_test.rb
```

## ファイル構成

- `alpaca.rb` - Alpacaクラスの実装
- `alpaca_test.rb` - コード品質チェック用のテストファイル
- `alpaca.jpg` - アルパカの画像
